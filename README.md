# 3-Statement-Model-Ratio-Analysis

# Model — Ultratech

## Overview

This repository contains a fully linked **3-Statement Financial Model** built in Excel for Ultratech.
The model forecasts financial performance using an integrated structure connecting:

* Income Statement
* Cash Flow Statement
* Balance Sheet

---

## Model Structure

The Excel file contains:

### 1. Historical Base

* Previous Year Balance Sheet used as the starting point

### 2. Financial Statements

* Forecast Income Statement
* Cash Flow Statement (Indirect Method)
* Forecast Balance Sheet

### 3. Supporting Schedules

* Debt Schedule
* Depreciation & Capex Schedule
* Shareholders’ Equity Schedule
* Working Capital Schedule

---

## Key Features

* 5-Year Forecasting Framework
* Automated Cash Flow Build
* Integrated Debt & Interest Calculations
* Depreciation Linked to Capex
* Working Capital Driven by operations
* Balance Sheet auto-balances (Assets = Liabilities + Equity)
